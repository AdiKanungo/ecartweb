## 📞 PRODUCT MANAGEMENT

### Updating Products
To update a product, modify the `productDatabase` in `app.js`:

```javascript
{ 
    id: 1, 
    name: 'Product Name', 
    category: 'Category', 
    price: 2499, 
    rating: 4.9, 
    reviews: 156, 
    image: 'filename.jpg', 
    description: 'Product description' 
}
```

### Adding New Products
Insert new product object at the end of the array before the closing bracket.

### Removing Products
Delete the entire product object from the array.
